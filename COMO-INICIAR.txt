┌──────────────────────────────────────────────────────────────────┐
│                                                                  │
│  OPCIÓN 1: CON DOCKER (MÁS FÁCIL - RECOMENDADO) ⭐              │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘

  1️⃣  Asegúrate de tener Docker instalado

  2️⃣  Abre una terminal en la carpeta del proyecto

  3️⃣  Ejecuta:

      docker compose up --build

      (Nota: es "docker compose" con espacio, no "docker-compose")

  4️⃣  Espera a que termine (primera vez tarda más)

  5️⃣  Abre tu navegador en:

      http://localhost:5000

  6️⃣  Inicia sesión con:

      Usuario:    admin
      Contraseña: Admin123!
